This codelab will walk you through how to setup Visual Studio Code for Java Programming using the Chocolatey installation manager. You will be installing the followings:

  • MS Visual Studio Code (the source code editor)
  • VS Code Extension Pack for Java.
  • Some other VS Code extensions, such as Colonize, Rainbow Bracket, etc). They are optional, but it is good to have (for better coding experience in VS Code).
  • OpenJDK (for Java Development)

This installation is only for Windows machine.

Chocolatey is a third-party installation manager tool for Windows. So we need to install it first.

Open a Poweshell Terminal as Administrator

  • Right click the Windows Icon. to open Windows menu.
  • Click on Windows Powershell(Admin)

Install Chocolatey

  • In the Powershell Terminal, run the following commands:
    Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
  • Test chocolatey installation
    choco --version

# If the chocolatey successfully installed, you should see the version.

In this step, we are going to install the following softwares:

  • Visual Studio Code (Code Editor)
  • Oracle OpenJDK (for Java Development)
choco install vscode oraclejdk -y

Open A New Powershell Session

The above installations may have changed the Environement Variables of your system such as the PATH. To allow the changes to take effect, open a new powershell session. Then in the following steps, you should work on the new powershell terminal.

In this step onwards, you are expected to work on the newly opened Powershell terminal. You can install VS Code Extensions from the VS Code. However, in this step we are going to do that using command-line in powershell terminal.

Extenstions to install

  • Extension Pack for Java
  • Live Share (for collaborative coding)
  • Error Lens (Show error at coding-time)
  • Colonize (add semicolon)
  • Better Comments (comments with colors)

Run the following commands in Powershell one line at a time.

 # Extension Pack for Java
 code --install-extension vscjava.vscode-java-pack

 # Live Share
 code --install-extension ms-vsliveshare.vsliveshare

 # Error Lens
 code --install-extension PhilHindle.errorlens
 
 # Colonize
 code --install-extension vmsynkov.colonize

 # Better Comments
 code --install-extension aaron-bond.better-comments
  • Launch VS Code. You can do this from powershell terminal, type the following command:
code
  • In VS Code, open the Command Palette by pressing the key Ctrl Shift P
  • In the text bar search for Java: Create Java Project...
  • Select the project type as No build tools
  • Select the project folder (i.e. where you want to save the project)
  • Name the project

    The project name will be used to create a folder for the project.

  • You should get a template project with the following folder structure.
  • Now, you can run the program by clicking on the Run button then Run Java or Debug Java, or simply press the key F5.
