This codelab will walk you through how to setup Visual Studio Code for OpenCV and Python Programming using the Chocolatey installation manager. You will be installing the followings:

  • MS Visual Studio Code (the code editor)
  • Python (the interpreter)
  • OpenCV package for Python (the library for Computer Vision)
  • VS Code Extension for Python (for python programming in VS Code)
  • VS Code Extension for Jupyter Notebook (for interactive document)
  • Some other VS Code extensions, such as Error Lens, Better Comments, etc). They are optional, but it is good to have (for better coding experience in VS Code).

This installation is only for Windows machine.

Chocolatey is a third-party installation manager tool for Windows. So we need to install it first.

Open a Poweshell Terminal as Administrator

  • Right click the Windows Icon. to open Windows menu.
  • Click on Windows Powershell(Admin)

Install Chocolatey

  • In the Powershell Terminal, run the following commands:
    Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://community.chocolatey.org/install.ps1'))
  • Test chocolatey installation
    choco --version

# If the chocolatey successfully installed, you should see the version.

In this step, we are going to install the following softwares:

  • VS Code
  • Python
choco install vscode python -y

Open A New Powershell Session

The above installations may have changed the Environement Variables of your system such as the PATH. To allow the changes to take effect, open a new powershell session. Then in the following steps, you should work on the new powershell terminal.

Test Python installation

python --version

# If python has been successfully installed, you should see the version.

In this step onwards, you are expected to work on the newly opened Powershell terminal. Run the following command in Powershell terminal

pip install  opencv-contrib-python

You can install VS Code Extensions from the VS Code. However, in this step we are going to do that using command-line in powershell terminal.

Extenstions to install

  • Extension for Python
  • Error Lens (Show error at coding-time)
  • Colonize (add semicolon)
  • Better Comments (comments with colors)
  • Path Intellisense (Filename autocomplete)
  • Live Share (for collaborative coding) Run the following commands in Powershell one line at a time.
 # Python
 code --install-extension ms-python.python

  # Error Lens
 code --install-extension PhilHindle.errorlens
 
 # Colonize
 code --install-extension vmsynkov.colonize

 # Better Comments
 code --install-extension aaron-bond.better-comments

 # Path Intellisense
 code --install-extension christian-kohler.path-intellisense

 # Live Share
 code --install-extension ms-vsliveshare.vsliveshare
